# Feature Ideas

## Data / IO

- Export to GeoJSON/KML/GPX file download
- Import GeoJSON by file drop or paste
- Share via URL (encode feature collection in URL hash or short link)
- Copy coordinates on click

## Editing

- Property editor panel — edit name, description, stroke, fill on selected features
- Snap to grid / snap to other features when drawing
- Duplicate selected feature
- Lock/unlock features to prevent accidental edits
- Measurement tool — show distance (line) or area (polygon)

## Map

- Layer opacity controls
- Toggle 3D terrain on/off
- Drawing on the minimap / overview map
- Geolocation "fly to me" button (userLocation already tracked)
- Ruler / scale bar

## Collaboration / Polish

- Dark mode for the UI
- Keyboard shortcut overlay (help modal)
- Feature list sidebar — click to select/zoom, reorder via drag
- Toast notifications for actions ("Feature deleted", "Exported 3 features")
